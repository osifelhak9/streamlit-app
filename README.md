# Loan Approval Prediction App

Application web de prédiction d'approbation de prêt utilisant le Machine Learning.

## Fonctionnalités

- Exploration interactive des données
- Visualisations avec Plotly
- Analyse des revenus et montants de prêts
- Simulation de décision de prêt

## Technologies

- Python
- Streamlit
- Pandas
- Plotly
- Scikit-learn
